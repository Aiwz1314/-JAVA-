### 2、结果填空（签到题）
1200000有多少个正约数
答案 96
```java
public class Main2 {

    public static void main(String[] args) {
        int sum = 2;
        for (int i = 2; i < 1200000; i++) {
            if (1200000 % i == 0) sum++;
        }
        System.out.println(sum);
    }
}
```
